# 📊 Marketing Funnel & Conversion Analysis Project


## 🟣 📌 Project Overview
This project analyzes customer conversion behavior using Power BI.  
It focuses on understanding marketing funnel performance, customer segmentation, and conversion trends.


## 🟡 📁 Dataset
**Bank Marketing Dataset**

Includes:
- Age  
- Job  
- Marital Status  
- Education  
- Contact type  
- Campaign outcome (y: yes/no)

👉 Target Variable: **Conversion Status (y)**



## 🔵 🛠 Tools Used
- Power BI Desktop 📊  
- Data Visualization  
- DAX Calculations  
- Data Cleaning & Transformation  



## 🟢 📊 Dashboard Features

### 📌 1. Conversion Analysis
Shows **Converted vs Not Converted customers**


### 📌 2. Job-wise Analysis
Displays customer distribution across job roles


### 📌 3. Customer Segmentation
Analyzes demographic patterns like education & marital status


### 🟠 KPI Card
- Total Customers (Count of records)


## 🟣 💡 Key Insights
- Majority of customers did not convert  
- Job roles like admin & technician show higher engagement  
- Conversion varies based on customer profile  
- Targeted marketing can improve conversion rates  


## 🔴 🎯 Conclusion
This analysis helps improve marketing strategies by identifying key customer segments and optimizing conversion performance  in the funnel.
